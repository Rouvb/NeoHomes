# NeoHomes
**NeoHomes** is a lightweight, GUI-based homes plugin for Paper servers, allowing players to manage multiple homes.
## Features
- Multiple homes per player
- GUI-based home selection
- Support SQLite storage
- Simple and clean commands
## Requirements
- Java 21 or higher
- Paper (or Paper-compatible) server
- No external dependencies required
## Commands
| Command            | Permission         |
|--------------------|--------------------|
| `/sethome <home>`  | `neohomes.sethome` |
| `/home [home]`     | `neohomes.home`    |
| `/delhome <home>`  | `neohomes.delhome` |
| `/neohomes reload` | `neohomes.reload`  |
